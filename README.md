# Project Overview
The objective of this task is to analyze e-commerce sales data and design an interactive Power BI dashboard that provides meaningful insights into sales performance, profitability, customer behavior, product categories, and payment methods.
The dashboard helps stakeholders understand business trends, identify profitable areas, and improve decision-making using data visualization.
